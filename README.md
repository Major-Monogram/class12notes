# Static version of my Blogspot Notes Blog

This is the GitHub repository for my static website! This project hosts a clean, fast, and responsive static version of the notes, guides, and important files from my personal Blogspot. 

It is automatically deployed and hosted via **GitHub Pages**.

Files are uploaded and maintained on a Google Drive and hyperlinks for the same are used on each notes page.

You can access the live site here: **https://major-monogram.github.io/class12notes/**

##  Repository Structure

The core architecture of this static site is built using minimalist, lightweight web technologies:

*   **`index.html`** – The main entry point of the website, containing the structured layout, navigation, and the core directory of hosted notes.
*   **`style.css`** – The stylesheet handling the visual design, ensuring a clean, readable, and responsive user interface across desktop and mobile devices.
